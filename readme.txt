All-in-one WooCommerce SEO plugin with AI meta descriptions, rich snippets, SEO analysis, Google preview, social cards, and sitemap optimization.

== Description ==

WooSEO Optimizer is a free, all-in-one WooCommerce SEO plugin that gives you premium-level SEO features without the premium price tag.

**Features:**

* **AI Meta Descriptions & Titles** — Auto-generate SEO-optimized titles and meta descriptions using OpenAI.
* **Product Rich Snippets** — Automatic JSON-LD schema with price, availability, ratings, brand, and GTIN.
* **Product SEO Analysis** — Real-time SEO scoring with 12+ checks for every product.
* **GTIN/Brand/Manufacturer Fields** — Add product identifiers directly in the WooCommerce editor.
* **Google Search Preview** — See how your product will appear in Google search results.
* **Social Media Cards** — OpenGraph, Twitter Card, and Pinterest-ready meta tags.
* **Enhanced Breadcrumbs** — Customizable breadcrumbs with Schema.org markup.
* **XML Sitemap Optimization** — Auto-hide product variations from your sitemap.
* **Variant Global Identifiers** — Per-variation GTIN/EAN/UPC/MPN fields with schema output.

== Installation ==

1. Upload the `wooseo-optimizer` folder to `/wp-content/plugins/`.
2. Activate through the 'Plugins' menu in WordPress.
3. Go to **WooSEO** in the admin sidebar to configure.
4. (Optional) Add your OpenAI API key for AI-powered features.

== Frequently Asked Questions ==

= Do I need an OpenAI API key? =
Only for the AI meta description generator. All other 8 features work without any API key.

= Does this work with Yoast SEO or RankMath? =
Yes. The plugin is designed to work alongside other SEO plugins. It adds WooCommerce-specific features they may not cover.

= Will this slow down my site? =
No. Each feature can be toggled on/off individually. Only active features load.
